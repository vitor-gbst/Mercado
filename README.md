# 🏪 Estoque - Sistema de Estoque

Este é um sistema simples para gerenciamento de estoque, onde é possível cadastrar, listar, entrade e saída de produtos.

## 📌 Funcionalidades
- 📦 Cadastrar produtos
- 📃 Listar produtos disponíveis
- 🛒 Entrada e saída de produtos
